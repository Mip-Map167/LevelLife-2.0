# LevelLife

<img width="286" height="287" alt="image" src="https://github.com/user-attachments/assets/46ac4a93-444e-4b40-b222-dffbb751fdc9" />

Система мотивации с игровыми элементами. Десктопное приложение на C# (Windows Forms), предназначенное для повышения продуктивности через геймификацию.

## Назначение

Приложение помогает структурировать повседневные задачи и отслеживать личный прогресс. За каждую выполненную задачу пользователь получает опыт, а при накоплении достаточного количества опыта повышается уровень персонажа. Таким образом, рутинные дела превращаются в игровой процесс с измеримыми результатами.

## Функциональные возможности

- Создание, редактирование и удаление задач с указанием награды в опыте
- Отметка выполнения задачи с автоматическим начислением опыта
- Система уровней с формулой расчёта необходимого опыта
- Отображение текущего прогресса (уровень, опыт, заполнение шкалы до следующего уровня)
- Ведение истории удалённых задач
- Статистика по задачам и накопленному опыту
- Автоматическое сохранение данных с созданием резервной копии
- График выполненных задач за последние семь дней
- Анимация получения опыта и повышения уровня

## Технологии

- Язык программирования: C#
- Платформа: .NET 6.0 / 8.0
- Графический интерфейс: Windows Forms
- Хранение данных: JSON (Newtonsoft.Json)
- Визуализация графиков: System.Windows.Forms.DataVisualization

## Структура проекта

- MainForm.cs — главное окно и логика интерфейса
- Task.cs — модель задачи
- Player.cs — модель игрока
- DataService.cs — сохранение и загрузка данных
- TaskDialog.cs — диалог добавления и редактирования задач
- GreetingDialog.cs — окно приветствия
- ConfirmDialog.cs — диалог подтверждения удаления

## Системные требования

- Операционная система Windows 10 или выше
- .NET Runtime 6.0 или выше
- 2 ГБ оперативной памяти
- 100 МБ свободного места на диске
