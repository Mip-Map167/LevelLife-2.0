# LevelLife — система мотивации с игровыми элементами

<img width="95" height="113" alt="image" src="https://github.com/user-attachments/assets/cb9f0e7f-2388-47ed-abfe-ed8205bfba73" />


Десктопное приложение на C# (Windows Forms) для повышения продуктивности через геймификацию: выполняешь задачи — получаешь опыт и повышаешь уровень персонажа.

## Возможности
- Добавление, редактирование, удаление задач
- Выполнение задачи → начисление XP
- Автоматическое повышение уровня с уведомлением
- Визуальный прогресс-бар и статистика
- Сохранение/загрузка данных в JSON (автосохранение)
- Тёмная тема, неоновые акценты, аватар персонажа

## Технологии
- .NET Framework 4.8 / .NET 6+
- C# / Windows Forms
- Newtonsoft.Json
